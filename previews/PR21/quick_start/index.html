Quick start

Suppose we want to sum a vector, which is the result of multiple broadcast expressions: y = x .+ x, z = 2 .* y, and, finally, sum(z). Let's write this in a function, foo, and compare the result of using lazy_broadcast to compute the intermediate values.

using Base.Broadcast: materialize
using LazyBroadcast: lazy_broadcast
using BenchmarkTools

function foo(x)
   y = x .+ x
   z = 2 .* y
   sum(z)
end;

trial = @benchmark foo(v) setup=(v=rand(10))
show(stdout, MIME("text/plain"), trial)
println()

function bar(x)
   y = lazy_broadcast.(x .+ x)
   z = lazy_broadcast.(2 .* y)
   sum(z)
end;

trial = @benchmark bar(v) setup=(v=rand(10))
show(stdout, MIME("text/plain"), trial)
println()
